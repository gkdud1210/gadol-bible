# 가돌 (Gadol Bible)

6개 언어 병렬 성경 읽기 PWA — 한국어·영어·중국어·러시아어·일본어·스페인어

---

## ✅ 현재 완료된 기능

### 홈 (index.html)
- **로그인 / 회원가입** 모달 (localStorage 기반)
- 게스트 없이 가입 후 bible.html로 이동
- 다크/라이트 테마 토글
- 홈 화면 "가돌" 타이틀 표시

### 성경 읽기 (bible.html)
- **6개 언어 병렬 표시**: 개역한글(KRV) · WEB 영어 · 和合本(중국어) · 시노달(러시아어) · 口語訳(일본어) · 레이나발레라(스페인어)
- 헤더 언어 토글 버튼 (한 EN 中 RU 日 ES) — 개별 활성/비활성
- **각 구절마다 언어별 🔊 TTS 버튼** — Web Speech API로 해당 언어 음성 재생
  - 재생 중 버튼 펄스 애니메이션 + ⏸ 표시
  - 하단 네비 "듣기" 버튼으로 전체 장 순서 읽기 (다시 누르면 중단)
- **단어 탭 기능** — 클릭 시 바텀 시트에 발음기호·뜻 표시
  - 🇨🇳 중국어: 병음(拼音) + 성조 안내 (ā á ǎ à)
  - 🇷🇺 러시아어: 강세 발음기호 (대문자=강세음절) + 키릴문자 발음 안내
  - 🇯🇵 일본어: 히라가나 읽기 + 로마자 표기 + 발음 안내
  - 🇪🇸 스페인어: 강세 발음기호 + 발음 안내
  - 🇺🇸 영어: 기본형 표시 + 어원·참고
  - 🇰🇷 한국어: 백과사전 패널 (Wikipedia 요약 + OSM 지도)
- 사이드바 — 구약/신약 66권 전체 목록 (그룹·접기/펼치기)
  - 현재 장 자동 펼치기 + 활성 표시
  - 아직 준비 중인 장은 `unavailable` 스타일 + 토스트 안내
- 스크롤 진행 바 (상단)
- 이전·다음 장 이동 (키보드 ← →)
- 단어 강조 모드 (하단 네비 🔤)
- 다크/라이트 테마 토글
- 로그아웃 버튼

## 📂 파일 구조

```
index.html        홈 — 로그인/회원가입
bible.html        성경 읽기 메인
credits.html      저작권 정보
manifest.json     PWA 매니페스트
sw.js             서비스워커 (캐시 전략)
css/
  main.css        전체 스타일 (다크·라이트, 6개 언어 색상, TTS 버튼, 발음기호)
js/
  data.js         BOOK_LIST(66권) + BIBLE 데이터 (창세기1, 요한복음1)
  dict.js         EN_DICT · KO_DICT · ZH_DICT · RU_DICT · JA_DICT · ES_DICT
  app.js          앱 로직 (렌더링·TTS·단어시트·사이드바·테마)
  bible-data.js   (추후 확장용 원시 데이터)
  word-data.js    (추후 확장용)
```

## 📌 진입점 URI

| 경로 | 설명 |
|------|------|
| `/index.html` | 홈 / 로그인·회원가입 |
| `/bible.html?book=GEN&ch=1` | 창세기 1장 (로그인 필요) |
| `/bible.html?book=JHN&ch=1` | 요한복음 1장 |
| `/credits.html` | 저작권 정보 |

## 🗂️ 데이터 모델

### BOOK_LIST (js/data.js)
```js
{ id, ko, en, ch, testament:'OT'|'NT', group }
```
- 66권 전체 (구약 39 + 신약 27)
- 현재 성경 본문 데이터: 창세기 1장(31절) + 요한복음 1장(51절)

### BIBLE (js/data.js)
```js
BIBLE['GEN_1'] = [{ v, ko, en, zh, ru, ja, es }, ...]
```

### 사전 (js/dict.js)
| 사전 | 키 형식 | 주요 필드 |
|------|---------|-----------|
| KO_DICT | 한국어 단어 | sections, wiki, osm |
| EN_DICT | 소문자 영어 | pos, ko, lemma, note |
| ZH_DICT | 중국어 어구 | ko, pinyin, note |
| RU_DICT | 소문자 러시아어 | ko, pron(강세), note |
| JA_DICT | 히라가나/한자 | ko, pron(読み方), note |
| ES_DICT | 소문자 스페인어 | ko, pron(강세), note |
